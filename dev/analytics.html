DayPal Analytics Dashboard

This dashboard explains and tests the analytics model used by the static hosted config pages.

What this can and cannot measure

This static dashboard reads analytics events saved in this browser's localStorage. It is useful for QA and for validating event names, but it is not a centralized production analytics service. Real cross-user totals require a backend endpoint or a hosted analytics provider.

Definitions

Unique user First known open from a browser/device identity.

Repeat user A later open from the same browser/device identity.

Geolocation Whether location mode is current device location or manual location. DayPal should not store exact GPS coordinates in analytics.

Manual location User enabled manual weather location by ZIP/postal code or city.
